# Node-RED Tips
Node-RED　環境構築

※構築環境のOSはUbuntu18.04

※ベース環境やアップデートにより版数やコマンドが変わる可能性あります

----
# ■Node-REDインストール
$ sudo apt-get update

$ sudo apt-get install npm

$ sudo npm install -g node-red

$ node -v

→v8.10.0

$ node-red

$ firefox http://localhost:1880

or

$ google-chrome http://localhost:1880
